# Tender Web Application

## Project Description
Tender is a multi-page business website that showcases company services, gallery, contact information, and client testimonials. The site is fully responsive and works on desktop, tablet, and mobile devices.

## Pages Included
1. Home
2. About
3. Services
4. Gallery
5. Contact
6. FAQ
7. Testimonials
8. Blog
9. Privacy

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Git & GitHub

## JavaScript Features
1. **Form Validation** - Validates contact form inputs (name, email, message) before submission
2. **Dark Mode Toggle** - Switches between light and dark theme

## How to Run the Project
1. Clone the repository: `git clone https://github.com/Nlidigu/tender.git`
2. Open `Home.html` in any web browser
3. Or use Live Server in VS Code

## Responsive Design
The website uses Bootstrap 5 grid system and media queries to ensure proper display on:
- Desktop (1200px and above)
- Tablet (768px to 1199px)
- Mobile (below 768px)
